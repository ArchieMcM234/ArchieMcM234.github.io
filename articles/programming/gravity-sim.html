Simple gravity simulation to learn C++

I have been meaning to learn C++ for a while as it seems to be a nice all rounder: some low level features, speed and hard-typed as oppose to Python's occasionaly overly soft-typed nature.

To start with I followed the W3 schools tutorials, which covered the basics but left me wanting with respect to classes and pointers. Then I descided to actually make something so I followed the first few videos of this SFML series and created this very basic gravity simulation.
